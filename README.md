# Thermal Doppler Broadening of Spectral Lines

## Overview

This project simulates the thermal Doppler broadening of spectral lines in a gas of atoms.

Atoms in a gas possess thermal velocities described by the Maxwell–Boltzmann distribution.  
These velocities produce Doppler shifts in emitted or absorbed photon frequencies, resulting in a broadened spectral line.

## Physics

### Maxwell velocity distribution

f(v) = sqrt(m/(2πkT)) exp(-mv²/(2kT))

### Doppler shift

ν = ν₀ (1 + v/c)

### Doppler width

Δν_D = (ν₀/c) √(2kT/m)

Thus increasing temperature increases atomic velocities and broadens the spectral line.
